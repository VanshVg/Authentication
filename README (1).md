
# Authorization

In auhtorization user needs to be authenticated to be able to update their username or delete their profile.



- The project uses:
    * Node.Js
    * Express.Js
    * Mongoose.Js
    * jsonwebtoken
## Installation

To install dependencies/packages 

```bash
  npm install
``` 

## Deployment

To deploy this project

```bash
  npm run dev
```
